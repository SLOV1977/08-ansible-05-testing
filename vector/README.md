Vector
=========

Ansible-роль для установки и настройки Vector

Переменные используемые ролью
-----------------------------

| Переменные | Описание |
|---------------|-----------------|
| vector_version | Версия Vector для установки |

Пример использования роли
-------------------------

    - hosts: vector
      roles:
         - vector-role
